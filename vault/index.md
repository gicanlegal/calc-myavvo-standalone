# Knowledge Base Index

Acest fișier este punctul de intrare pentru orice agent care lucrează în proiect.
Citește-l primul la începutul fiecărei sesiuni.

## Proiect
- Nume: `calc-myavvo-standalone`
- Tip: aplicație React + TypeScript + Vite
- Scopul acestui vault: memorie persistentă pentru context de proiect, decizii, sesiuni și concepte.

## Structură
- [[raw/]] - date brute, note neprocesate, capturi de context
- [[wiki/index]] - knowledge base procesată
- [[log]] - istoric operațiuni și schimbări importante
- [[CLAUDE]] - reguli de lucru pentru agenți

## Topicuri principale

## Utilizare pentru agenți
1. Citește acest fișier la începutul sesiunii.
2. Citește apoi `vault/CLAUDE.md`.
3. Dacă există context relevant, navighează în `vault/wiki/`.
4. Când apar decizii, buguri, convenții sau pași importanți, salvează-i în wiki.
5. Actualizează indexurile când adaugi informații noi.
