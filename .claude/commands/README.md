# Slash commands locale pentru proiect

Aceste comenzi sunt disponibile la nivel de proiect în `.claude/commands/`.

## Comenzi recomandate
- `/start-session` — citește contextul principal al proiectului și face onboarding rapid
- `/memory` — rulează rutina completă de memorie: citește, folosește, persistă
- `/update-memory` — actualizează wiki-ul după modificări importante
- `/check-memory` — verifică starea curentă a memoriei și rulează lint
- `/close-task` — închide taskul, rezumă schimbările, actualizează memoria și verifică statusul repo-ului

## Scop
Să nu mai fie nevoie să spui manual agentului de fiecare dată să:
- citească wiki-ul
- verifice ce există deja
- actualizeze wiki-ul și log-ul la final

## Limită
Aceste comenzi sunt definite local pentru acest proiect și trebuie să opereze doar în cadrul acestui repo.
