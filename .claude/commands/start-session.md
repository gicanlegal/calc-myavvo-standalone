# Start Session — Inițializează sesiunea cu memoria proiectului

Rulează rutina de onboarding pentru acest proiect.

## Pași
1. Citește `vault/index.md`
2. Citește `vault/CLAUDE.md`
3. Citește `vault/wiki/index.md`
4. Citește paginile principale:
   - `vault/wiki/concepts/project-overview.md`
   - `vault/wiki/concepts/architecture.md`
   - `vault/wiki/concepts/testing-strategy.md`
   - `vault/wiki/concepts/ci-cd-and-quality-gates.md`
5. Dacă există, consultă și ultimul fișier din `vault/wiki/daily/`

## Apoi
Afișează un rezumat scurt cu:
- ce este proiectul
- arhitectura de bază
- statusul calității (tests / CI / husky)
- ce context persistent există deja în wiki
- întreabă utilizatorul ce vrea să facă mai departe

## Reguli
- Lucrează doar în limitele acestui proiect.
- Nu folosi informații din alte repo-uri.
- Dacă descoperi informații noi importante în această sesiune, persistă-le în `vault/` la final.
