# Check Memory — Verifică rapid starea memoriei proiectului

## Ce trebuie să faci
1. Citește:
   - `vault/index.md`
   - `vault/wiki/index.md`
   - `vault/log.md`
2. Rulează `python3 memory/lint.py`
3. Rezumă:
   - topicurile principale existente
   - ultimele actualizări din log
   - dacă există broken links / orphan pages / alte probleme
   - ce zone din wiki par incomplete

## Reguli
- Nu modifica nimic decât dacă utilizatorul cere explicit.
- Lucrează doar în acest proiect.
